# Finance Tracker Frontend

A modern and responsive Finance Tracker Web Application frontend built with React.js and Tailwind CSS.

## Features

- User Signup with OTP Verification
- User Signin
- Forgot Password with OTP
- Change Password
- Protected Routes using JWT
- Add Transactions
- View Transactions
- Filter Transactions
  - By Type
  - By Date Range
  - By Amount Range
- Sort Transactions
  - By Date
  - By Amount
  - By Type
- Pagination
- Profile Page
- Logout

## Tech Stack

- React.js
- React Router DOM
- Axios
- Tailwind CSS
- React Hot Toast

## Installation

```bash
git clone <https://github.com/mohdaneesh018/-Finance_Tracker_Web_Application_Frontend>
cd frontend
npm install
npm run dev

Environment Variables:

Create .env file:

VITE_API_URL=http://localhost:5000/api
